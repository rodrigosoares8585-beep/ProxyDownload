# 🤖 Como Adicionar o Mascote de Fundo

## 📁 Localização do Arquivo

Você deve colocar a imagem do mascote em:
```
/home/rodrigo/bot_local/src/Site Dowloand de Audio Proxy/public/images/mascote.png
```

## 🖼️ Requisitos da Imagem

- **Nome:** `mascote.png`
- **Formato:** PNG, JPG, WEBP (cualquer formato de imagem)
- **Tamanho recomendado:** 1920x1080px ou maior
- **Resolução:** Mínimo 1080p (Full HD)
- **Peso:** Até 2MB para melhor performance

## 🎨 Características do Design

O mascote será exibido com:
- ✨ **Overlay Gradiente:** Efeito semi-transparente para melhor legibilidade
- 🌊 **Fixed Background:** O mascote fica fixo enquanto você scroll
- 🎭 **Modo Dramático:** Mascote gigante cobrindo toda a tela
- 💫 **Animação Suave:** Overlay pulsante que anima suavemente
- 🔲 **Backdrop Blur:** Cards contêm blur para destacar o conteúdo

## 📸 Copiando a Imagem

### Opção 1: Via Terminal
```bash
cp /caminho/da/sua/imagem.png "/home/rodrigo/bot_local/src/Site Dowloand de Audio Proxy/public/images/mascote.png"
```

### Opção 2: Copiar Manualmente
1. Copie sua imagem em qualquer formato
2. Renomeie para `mascote.png`
3. Cole em: `Site Dowloand de Audio Proxy/public/images/`

## ✅ Verificar instalação

Depois de colocar a imagem, abra o site em:
```
http://localhost:3000
```

E você verá o mascote gigante como fundo! 🤖

## 🎯 Efeitos Visuais

- **Overlay:** 85% opacidade com gradiente roxo/azul
- **Blur:** 10px de efeito blur nos cards
- **Animação:** Pulsante suave a cada 10 segundos

## ❓ Problemas?

Se a imagem não aparecer:
1. ✅ Verifique se o arquivo está em `public/images/mascote.png`
2. ✅ Limpe o cache do navegador (Ctrl+Shift+Delete)
3. ✅ Reinicie o servidor Node.js
4. ✅ Verifique o console (F12) para erros

## 💡 Dica

Para melhor resultado, use uma imagem:
- Com foco na parte superior/central
- Com cores que harmonizem com roxo/azul
- Com transparência em volta (PNG) para efeito melhor
